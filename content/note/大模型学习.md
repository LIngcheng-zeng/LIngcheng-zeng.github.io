# 环境准备

## miniconda

```sh
# Create a new environment  创建一个新环境, --name 标志用于设置新环境的名称。
conda create --name hello-env python=3.14
# 先激活新创建的环境才能使用它
conda activate hello-env
```

## pipx

```sh
python -m pip install --user pipx
```

## poetry

```shell
# 安装poetry
pipx install poetry
# 初始项目
poetry init app-name 
# 根据配置文件，一键同步项目的开发环境
portry install
```

## 引入 openai 包

```
# 在以构建的项目项目下
poetry add openai
```

## 运行

```sh
poetry run python src/pythondemo/main.py
```

## 配置切换

```shell
python src/pythondemo/main.py resource=common_config
```

## 安装uv

```sh
pip install uv

uv pip install app-n
```
