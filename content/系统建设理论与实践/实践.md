# 目的

正确的 ，综合 ，概要的  评估所行动和所付出 带来的 效果

# 指标制定

## 业务数据指标 

### 正向指标

1. 每 X分钟   平台订单成交量
2. 每 X分钟   单商品订单成交量
3. 商品收藏收藏数量

### 反向指标

1. 用户退货数量 
2. 实际支付额度 与 预期商品支付价格的 差异

## 服务能力指标

1. 响应时间
2. 服务响应异常比例
3. 连接
   1. 任务处理线程池
   2. 各组件连接池
4. CPU / 内存 /磁盘 /网络带宽 占比
5. 链路强依赖接口服务质量
   1. 支付平台
   2. 用户服务

# 指标设计实现

## 业务数据指标实现

核心动作发生后，发送消息，通过消费消息，完成数据上报
